# Simulación de Red SOHO – Cisco Packet Tracer

## Descripción del Proyecto

Este proyecto consiste en el diseño e implementación de una red tipo SOHO (Small Office / Home Office) utilizando Cisco Packet Tracer.

La red integra tecnologías LAN, WLAN y WAN, incluyendo segmentación mediante VLANs, enrutamiento dinámico y servicios de red como DHCP, DNS, NAT y HTTP. Además, se implementa conectividad inalámbrica mediante el uso de un controlador WLC y un punto de acceso LAP.

La documentación técnica completa del laboratorio se encuentra en la sección Wiki de este repositorio.

---

## Objetivos Técnicos

- Diseñar una topología de red SOHO funcional.
- Implementar un esquema de direccionamiento IPv4 utilizando VLSM.
- Configurar VLANs para segmentación de red.
- Implementar enrutamiento inter-VLAN mediante router-on-a-stick.
- Configurar el protocolo de enrutamiento dinámico EIGRP.
- Implementar servicios de red: DHCP, DNS, NAT y HTTP.
- Validar la conectividad entre dispositivos de la red.
- Analizar el flujo de datos mediante simulación en Packet Tracer.
- Documentar el desarrollo del laboratorio en GitHub Wiki.

## Características de la Red

- Segmentación lógica mediante VLANs (30, 50, 65 y 99)
- Subneteo mediante VLSM
- Asignación dinámica de direcciones IP
- Acceso a servidor web mediante dominio: des.net

- Integración de red cableada e inalámbrica
- Acceso remoto a dispositivos mediante Telnet
